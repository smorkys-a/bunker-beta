///////Bunker Game (C++)\\\\\\\\

Console-based survival game for 6 players written in C++.
Project Description:
"Bunker" is a multiplayer console game where 6 players receive random hidden characteristics such as:
Profession, Health condition, Hobby, Phobia, Item, Personality trait, Age

During several rounds, players reveal information, vote to eliminate others, and try to build the best possible team to survive inside a bunker after a global catastrophe.

The final team is evaluated based on:

Required skills (medical, engineering, food, security, etc.)

Available bunker resources (food, water, medicine)

Health risks

Random events

How to Run:

Compile using g++:
g++ main.cpp -o bunker.exe -Wl,-subsystem,console

Then run:
.\bunker.exe
